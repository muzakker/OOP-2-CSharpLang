Address must contain house number, street name & upazila name; id can have max value of 300 as it is a very small company. There will be a method, which will count the yearly salary of every employee. Another method will break down the yearly salary structure of every employee where 50% of his/her yearly salary is basic salary, 30% house rent, 15% medical allowance and 5% will be cell phone bill. Now a Show Info method will provide all the above-mentioned information as output.

Create 2 object, initialize first one with default constructor and take the values of the field as input. For second object use-parameterized constructor for initialization (no need to take input here). Now print all info.
